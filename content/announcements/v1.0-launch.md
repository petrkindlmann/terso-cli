## LinkedIn

```
We shipped Terso CLI v1.0 today.

It does one thing: compiles a single `AGENTS.md` into the per-agent config files
each AI coding assistant expects — CLAUDE.md, .cursorrules,
.github/copilot-instructions.md — so you maintain one file instead of three.

Offline. Free. MIT-licensed.

Two reasons we built it:

1. The drift problem is real. Teams that adopt 2+ AI coding agents end up with
   subtly inconsistent project rules across files. Reviewers can't catch it
   because each file looks fine in isolation.

2. AGENTS.md is the format AI tools are converging on. Codex CLI already reads
   it natively. The hope is that more agents follow. Until they do, Terso CLI
   makes the format sticky by reducing the cost of adopting it.

`npm install -g terso-cli` and you're going.

[link to repo]
```
